Flight Simulator
================

This program includes several functions:

  Scene control
-----------------------
- f: toggle fullscreen
- g: toggle fog
- s: toggle sea and sky
- w: toggle wire
- q: toggle quit
- t: toggle texture (though not perfect)
- m: toggle mountain
-----------------------
  Camera control
-----------------------
- Page up   : faster
- Page down : slower
- Up Arrow  : move up
- Down Arrow: move down
- Mouse Right: turn right
- Mouse left : turn left
-----------------------
